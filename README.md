Floripa Bus Schedule Application
================================

* This app searches for Bus Routes and Timetables
* Platform: Android
* Target API: 19
* Tested on a Samsung S4 Mini Smartphone (4.2.2) and a Motorola Xoom Tablet (4.1.2)
